## Project Overview 


This project focuses on data cleaning using power Query in Excel where i fixed inconsistent date format, spacing issues, missing values
and improved calculations by replacing manual revenue entries with SUMPRODUCT formula to improve accuracy. 

## Tools Used

- Microsoft Excel 
- Power Query

## Project Objectives 

* Standardized date format for easier analysis 

* Fixed spacing issues in column headers to improve consistency and readability

* Improved calculations by replacing manual revenue entries with a SUMPRODUCT formula to automate updates and improve accuracy

* Handled missing values by replacing nulls in the coupon column with "No Coupon".
